Icon

ProTrack icon component supports more than 150,000 different icons. Below are examples of some icon sets that can be used together without problems.

Material Symbols Icons (Default)

Material Symbols is the modern icon font consolidating over 3,275 glyphs with a wide range of design variants. It is a default font in the ProTrack.

Phosphor Icons

Phosphor icons – is a great icon pack that has 6 different styles per icon (thin,light,regular,bold,duotone,fill) and there are over 1300 icons in total.

Circle Flags Icons

Circle Flags is a collection of 400+ minimal circular SVG country, state and language flags.
